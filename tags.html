Important HTML tags

This page is designed as a quick reference for some of the common tags covered in Learn Enough HTML to Be Dangerous . In the process of making it, we'll learn hoe to make HTML tables via table and related tags.

The tables below don't include all HTML tags, but they do list many of the most important ones.

Block Elements

    Tag Name Purpose
  		1. h1–h6 headings include a heading (levels 1–6)
  		2. p paragraph include a paragraph of text
  		3. table table include a table
  		4. tr table row include a row of data
  		5. th table header make a table header
  		6. td table data include a table data cell

Inline Elements

    Tag Name Purpose Example Result
  		• em emphasized make emphasized text <em>technical sophistication </em> technical sophistication
  		• strong strong make strong text <strong>at least a billion people</strong> at least a billion people
  		• a anchor make hyperlink < a href="https://learnenough.com/" >Learn Enough</a> Learn Enough
  		• img image include image <img src="https://bit.ly/1MZAFuQ" alt="Michael Hartl">

Document Tags

    Tag Name Purpose
  		1. <!DOCTYPE html> html tag tells editor that this document is HTML
  		2. <head> head contains the webpage's title, as well as other information
  		3. <body> body contains all what you see in the webpage
  		4. <title> title gives the title of the webpage
  		5. <meta> meta enables to set character set

Other Tags

    Tag Name Purpose
  		• style style lets you add inline style to html
  		• div division divides contents into blocks
  		• span span seperates a code inline within a div
  		• blockquote blockquote turns a paragraph into "quote-like" appearance
  		• br line break breaks a sentence into a new line
